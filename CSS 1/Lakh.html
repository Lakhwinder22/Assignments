What is CSS?

  1. CSS stands For Cascading Style Sheets.

  2. CSS describes how HTML elements are to be displayed on screen,paper,or in other media.

  3. Css is not neccessaryCSS saves a lot of work.It can control multiple web pages all at once

  4. External stylesheet are stored in CSS files.

CSS can be added to HTML Document in 3 ways:

  • Inline - by using the style attribute inside HTML elements1
  • Internal - by using style element in the head section
  • External - by using a link element to link an element CSS file.
  • The most common way to add CSS, is to Keep the style in external CSS files. However, in this tutorial we will use inline and internal styles, because this is easier to demonstrate , and easier for you to try it yourself.

When Comparting CSS class vs ID, the difference is that CSS class applies a style to multiple elements on the other hand, applies a style to one unique element.ID is also special in that you use a special URL to link directly to an element and it's used by JavaScript.

In CSS, selectors are used to target a specific element or range of element in a web page. Once an element has been targeted, a style or set of style can be applied to the element.

There is a wide range of selectors available.Two of most Commonly used are class and ID.Both are used to target element to which a style should be applied.
